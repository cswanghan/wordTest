# 📊 记忆分析功能使用指南

## 功能概述

Lion Festival G3 Spelling Application 现在集成了强大的记忆分析功能，帮助用户深入了解自己的学习模式、记忆弱点和进步趋势。通过详细的数据收集和分析，用户可以：

- 📈 追踪学习进度和表现
- 🎯 识别记忆薄弱环节
- 📊 优化学习策略
- 💡 获得个性化学习建议

## 🎮 功能详解

### 一、练习后详细报告

每次完成练习后，用户将看到比以往更详细的分析报告：

#### 1.1 基础统计
- **总分**：本次练习获得的分数
- **准确率**：基于所有按键的正确率
- **最大连击**：连续完美单词的最高数量
- **总用时**：完成本次练习的总时间

#### 1.2 表现分析
- **表现最佳**：错误最少、用时最短的单词
- **需要改进**：错误较多或用时较长的单词
- **错误热力图**：可视化显示每个字母的错误频率
  - 🟢 绿色：一次正确
  - 🟡 黄色：1次错误
  - 🟠 橙色：2次错误
  - 🔴 红色：3次以上错误
- **热门错误**：最常见的字母混淆对

### 二、记忆分析中心

访问方式：
- 主页 → 用户仪表板 → 记忆分析中心
- 练习结果页 → 记忆分析按钮

#### 2.1 今日练习概览
显示当天练习的详细数据：
- **完成单词数**：今日总共练习的单词数量
- **完美单词数**：零错误完成的单词数量
- **准确率**：基于所有输入的正确率
- **平均用时**：每个单词的平均完成时间

**困难单词识别**：
系统自动识别错误率最高的单词，建议重点复习。

**热门错误统计**：
统计最容易混淆的字母对，例如：
- e → i (5次)
- o → a (3次)

#### 2.2 本周学习趋势
- **总单词数**：本周累计练习的单词数量
- **日均单词**：平均每天练习的单词数
- **平均准确率**：本周整体准确率
- **准确率变化**：与上周相比的准确率变化
- **速度变化**：与上周相比的速度变化

**进步趋势可视化**：
- 📈 准确率上升/下降百分比
- 🚀 速度提升/下降（毫秒）

#### 2.3 智能洞察
系统自动生成个性化建议：

**表现优秀**：
- 准确率 ≥ 90%
- 建议继续保持当前学习节奏

**需要改进**：
- 准确率 < 70%
- 建议放慢速度，专注准确性

**困难单词提醒**：
- 识别当日最困难的3个单词
- 建议针对性练习

**常见错误提醒**：
- 识别最频繁的字母混淆
- 建议注意区分相似字母

## 📊 数据收集详情

### 详细错误记录
每次错误输入都会被详细记录：
- **位置**：错误发生在单词的哪个位置（索引）
- **期望字母**：应该输入的正确字母
- **实际字母**：用户输入的错误字母
- **时间戳**：错误发生的精确时间
- **相邻字母**：错误位置的前后字母
- **字母类型**：元音、辅音或其他

### 错误模式分析
系统自动识别错误类型：
1. **键盘相邻键错误**：如 f → g（键盘相邻）
2. **发音相似错误**：如 b → p（发音相似）
3. **替换错误**：其他类型的字母替换

### 记忆强度评估
- **首次正确率**：第一次输入即正确的概率
- **学习曲线**：同一单词多次练习的表现变化
- **遗忘曲线**：间隔时间与错误率的关系

## 🎯 使用建议

### 每日练习后
1. 查看练习后报告，了解本次表现
2. 重点关注错误热力图，识别薄弱字母
3. 记录困难单词，课后重点练习

### 每周复盘
1. 访问记忆分析中心
2. 查看本周学习趋势
3. 对比上周数据，评估进步
4. 根据智能洞察调整学习策略

### 长期学习规划
1. 持续追踪困难单词
2. 重点练习高频错误字母
3. 监控准确率和速度趋势
4. 保持稳定的学习节奏

## 💾 数据存储

所有数据安全存储在用户本地浏览器中：
- **每日统计**：保留30天
- **每周统计**：保留1年
- **会话历史**：保留最近100次练习

用户可以随时：
- 查看自己的学习数据
- 导出所有数据为JSON文件
- 清除所有数据（不可恢复）

## 🔐 隐私保护

## ❓ 常见问题

**Q: 数据会丢失吗？**
A: 数据存储在浏览器本地，清除浏览器数据会丢失。建议定期导出数据。

**Q: 准确率如何计算？**
A: 准确率 = 正确按键数 / (正确按键数 + 错误按键数) × 100%

**Q: 为什么有些单词没有显示在困难单词中？**
A: 系统只显示练习2次以上的单词，避免偶然误差。

**Q: 可以查看历史数据吗？**
A: 可以，通过记忆分析中心可以查看每日和每周的历史数据。

**Q: 数据导出包含什么？**
A: 包含所有练习记录、统计数据、错误详情和操作日志。
